# SharePoint DownloadFile

Down load a file into a hyperlink :

```
    <a href="/Site/SubSite/Teamsite/_layouts/download.aspx?SourceUrl=http:///Site/SubSite/Teamsite/Documents/File.xlsx">Download A Copy</a>
```
