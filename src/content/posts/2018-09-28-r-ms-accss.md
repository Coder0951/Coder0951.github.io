# R MS Accss

## Problem

Query an MS Access Database

## Solution

```
    # Load RODBC package
    library(RODBC)
    # Connect to Access db
    channel<- odbcConnectAccess("\\\\ent.core.company.com\\City\\City\\Folder\\Folder\\File.accdb")
    # Get data
    data <- sqlQuery( channel , paste ('select *
    from "AQry" '))
```
